# zahnarzt-schopplich.de

## Usage

### Development

```bash
pnpm dev
```

Afterwards, visit [localhost:3333](http://localhost:3333).

### Build

To build the app, run:

```bash
pnpm build
```

The generated files will be put into `dist`, ready to be served.
